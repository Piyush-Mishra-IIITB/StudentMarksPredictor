A Machine Learning project that predicts a student's overall performance based on personal and academic details. Built with Python, scikit-learn, and Streamlit for an interactive, user-friendly web app.

---

## Features

- Predict student scores using:
  - Gender
  - Race/Ethnicity
  - Parental Level of Education
  - Lunch Type
  - Test Preparation Course
  - Reading and Writing Scores
- Uses the same preprocessing pipeline as training to ensure consistent predictions.
- Interactive UI built with Streamlit.
- Visualizations for better understanding of predictions.
